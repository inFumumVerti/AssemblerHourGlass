# AssemblerHourGlass
A hour glass timer, programmed in 8051 assembler code.

## Konfiguration

LED-Matrix: 
- Y-Achse: P0
- X-Achse: P1

Keypad:
- A: Start; P3.0
- B: Stop/Reset; p3.2

LED Panel:
- Countdown fertig: P3.7
